### Описание

Этот документ содержит визуальное представление всех эндпоинтов API, а также примеры запросов и ответов. Используйте изображения для быстрого ознакомления с форматами данных и функциональностью API.

---

### Как использовать

1. **Все эндпоинты**: Обзор всех доступных эндпоинтов API.
2. **Создание самолета**: Формат сообщения для создания нового самолета.
3. **Готовность самолета**: Формат сообщения о готовности самолета к регистрации.
4. **Запрос времени**: Формат запроса для получения текущего времени.
5. **Запрос таймаута**: Формат запроса для получения таймаута.
6. **Изменение секунд в тик**: Формат запроса для изменения количества секунд в тик.
7. **Настольное приложение**: Скриншот настольного приложения для управления временем.
